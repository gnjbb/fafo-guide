# fafo-guide

**an onboarding experience for fucking around and finding out.**

aka an ai-powered ideation workflow for claude code - enjoy the ride.

## installation

```bash
npx github:gnjbb/fafo-guide install
```

follow the instructions.

files are installed to `_fafo-guide/` in your current directory.

## usage

after installation:

1. open claude code in your project directory
2. say: **"run the fafo guide workflow"**
3. follow the prompts (click buttons, minimal typing required)

you'll get a **ship sheet** - a battle plan to ship a demo version with a failsafe plan if things go sideways
